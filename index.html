خصائص HTML مع الشرح

الخصائص (Attributes) في HTML تستخدم لإضافة معلومات إضافية إلى العناصر وتحدد كيفية عملها.

1. خاصية href

تستخدم في عنصر <a> لتحديد رابط الانتقال:

<a href="https://example.com">زر الانتقال</a>

2. خاصية src

تُستخدم في عنصر <img> لتحديد مسار الصورة:

<img src="image.jpg" alt="صورة">

3. خاصية alt

تُستخدم في عنصر <img> لتوفير نص بديل للصورة:

<img src="image.jpg" alt="وصف للصورة">

4. خاصية title

تعرض تلميحًا يظهر عند تمرير الفأرة فوق العنصر:

<p title="هذه فقرة">مرر فوقي

5. خاصية style

تستخدم لإضافة تنسيقات CSS مباشرة إلى العنصر:

<p style="color: red; font-size: 20px;">نص باللون الأحمر</p>

6. خاصية id

تُستخدم لإعطاء معرف فريد للعنصر:

<div id="unique">محتوى</div>

7. خاصية class

تُستخدم لتعيين فئة (class) للعنصر:

<p class="text-bold">نص غامق</p>

8. خاصية disabled

تعطّل زر أو حقل إدخال:

<input type="text" disabled>

9. خاصية readonly

تجعل حقل الإدخال غير قابل للتحرير:

<input type="text" value="لا يمكن التعديل" readonly>

10. خاصية checked

تجعل خيار التحديد مفعلاً بشكل افتراضي:

<input type="checkbox" checked>

11. خاصية placeholder

تعرض نصًا توضيحيًا داخل حقل الإدخال:

<input type="text" placeholder="أدخل اسمك">

12. خاصية required

تجعل الحقل مطلوبًا قبل الإرسال:

<input type="text" required>

13. خاصية maxlength

تحدد الحد الأقصى لعدد الأحرف:

<input type="text" maxlength="10">

14. خاصية min و max

تُستخدم مع الحقول العددية لتحديد القيم الدنيا والعليا:

<input type="number" min="1" max="10">

15. خاصية autocomplete

تسمح للمتصفح بإكمال القيم تلقائيًا:

<input type="text" autocomplete="on">
